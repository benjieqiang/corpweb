## 简介

> 该项目是国内著名教育培训机构一期的Python中Django实战课程，去年做的一个项目。实现了一个简单的企业官网。做得比较简陋，适合刚入门Django的人下载下来跑一下。
>
> 完成的功能：
>
> 1. 首页的轮播图显示；
> 2. 瀑布流的练习；
> 3. 自定义过滤器配合forloop实现
> 4. 按照权重（按从大到小排列）排序首页内容；

## 快速开始

### 运行环境

> - **Python Version：**Python3.5
> - **Django Version:** Django2.1
> - **开发工具：**Pycharm 2018
